Hoe werkt de koppeling?

1 De app haalt automatisch de boekingen van vandaag op uit MICE.

2 Selecteer de juiste boeking in de balk bovenaan — klantgegevens worden ingevuld.

3 Na het feest: tik op "Opslaan in MICE" op het drankjes-tabblad — het verbruiksoverzicht wordt als notitie opgeslagen bij de boeking in MICE.

4 Prijzen worden automatisch gesynchroniseerd via het Prijzen-tabblad.
API instellingen
De API key wordt beheerd via Vercel — veilig opgeslagen als omgevingsvariabele, nooit zichtbaar in de app. Stel hem in via:

Vercel → jouw project → Settings → Environment Variables

Variabele naam: MICE_API_KEY
